PacMan Exercise Repository

This exercise revolved around moving PacMan avross the screen, every 200 miliseconds and having him reverse when he hits the screens edge. Additionally aiding in our understanding how we can manipulate html elements with the DOM was an important part of the exercise. I control a PacMan image to simulate the PacMan moving across the screen and reversing direction. Also, being a good model of how the browser holds information in memory. I learnt to pass four agruments into setInterval as well as checking for page boundaries, both tricky at first but incredibly valuable skills.

Following Eyes Exercise Repository

This exercise focused on coding an additional eye to our screen that would follow the cursor of the user. Specifically, for this exercise I had to add a second "eye" element, and have that element follow the mouse position on the screen using mouse events. With mouse events being a popular way to bind the DOM elements to events, such as click, mousemove, & mouseleave. I found it an enjoyable exercise both interesting and practical./

Bus Stop Tracker Repository

My final project presented on this page is a real-time bus stop tracker for MIT's campus. For this exercise, I developed an animation that displays a city bus location in real-time. Involving three steps, where I refactored the project, shareed it on my GitHub repository, and wrote a discussion post describing my work. This utilized mapbox for my first time, an open-source platform that allows me to create and display maps on a web page. Using the MBTA bus data to determine stops between Harvard and MIT, then adding an animated marker on the map to highlight the bus routes.
